# Be aware!
(Only DNS related functionality is working atm)
-This tool uses on occasion the command "dig" for dns querries, so please have it up to date to avoid errors
-Type on the terminal: "node ." or "node stalkermap.js" inside the project directory into execute it
-All info gathered by the tool will be store on "./data/{targetName}" 
(WIP - to see the stored data take a look on "./data/appData/{targetName}")
